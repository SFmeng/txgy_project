🔧 API接口信息

权限管理API:

GET /api/v1/organization/permissions/manage/?group_by_module=true # 按模块分组获取权限 GET /api/v1/organization/permissions/manage/?page=1&page_size=20 # 分页获取权限列表 POST /api/v1/organization/permissions/manage/ # 创建新权限 PUT /api/v1/organization/permissions/{perm_id}/ # 更新权限 DELETE /api/v1/organization/permissions/{perm_id}/ # 删除权限

菜单管理API:

GET /api/v1/organization/menus/?terminal=pc # 获取菜单树 POST /api/v1/organization/menus/ # 创建新菜单 PUT /api/v1/organization/menus/{menu_id}/ # 更新菜单 DELETE /api/v1/organization/menus/{menu_id}/ # 删除菜单
