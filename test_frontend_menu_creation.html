🔍 问题排查

如果新菜单没有自动出现在侧边栏，可能的原因：

  • 用户角色信息没有正确加载
  • 自动分配逻辑没有执行
  • 菜单刷新机制没有触发
  • 角色菜单分配API调用失败

排查方法：

  • 打开浏览器开发者工具，查看Console日志
  • 查看Network面板，确认API调用是否成功
  • 检查用户Profile API是否返回了角色信息
  • 验证菜单分配API是否被正确调用

🛠️ 技术实现

自动分配逻辑：

// 在菜单创建成功后 const response = await request({ url: '/organization/menus/', method: 'post', data: menuForm }) // 自动分配新菜单给当前用户的角色 if (response.data && response.data.menu_id) { await autoAssignMenuToUserRoles(response.data.menu_id) } // 刷新全局菜单 menuStore.refreshMenus()

关键API调用：

  • GET /api/v1/auth/profile/ - 获取用户角色信息
  • GET /api/v1/organization/roles/{role_id}/ - 获取角色当前菜单
  • POST /api/v1/organization/assign/menus/ - 分配菜单给角色
  • GET /api/v1/organization/user/permissions/ - 刷新用户菜单
